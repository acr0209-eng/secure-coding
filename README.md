# Secure Market

중고거래 플랫폼 과제를 위한 Next.js 기반 웹 애플리케이션입니다.

## 주요 기능

- 회원가입, 로그인, 로그아웃
- 상품 등록, 상품 목록, 검색, 분류 필터
- 상품 상세 보기와 판매자 문의
- 안전거래 금고를 통한 사용자 간 송금
- 악성 상품/사용자 신고
- 관리자 상품 상태 변경, 사용자 차단, 신고 처리, 송금 장부 확인
- SQLite와 Prisma 기반 로컬 데이터베이스

## 과제 요구사항 매핑

| 요구사항 | 구현 위치 |
| --- | --- |
| 사람들이 플랫폼에 가입 | 회원가입, 로그인 |
| 상품을 올리고 볼 수 있음 | 상품 등록, 상품 목록, 상품 상세 |
| 사용자끼리 소통 | 상품 상세 문의 기능 |
| 악성 유저나 상품 차단 | 신고 기능, 관리자 사용자 차단, 상품 차단 |
| 유저 간 송금 | 안전거래 금고, 상품 상세 안전송금 |
| 상품 검색 | 키워드 검색, 카테고리 필터 |
| 관리자가 모든 요소 관리 | 관리자 대시보드의 사용자, 상품, 신고, 송금 장부 |

## 독창성 포인트

- 안전거래 금고: 플랫폼 내부 지갑 잔액으로 사용자 간 송금
- 상품 연동 송금: 상품 가격 이상을 판매자에게 보내면 자동으로 거래완료 처리
- 신고 레이더: 신고 내역을 관리자 대시보드에서 바로 확인하고 조치
- 송금 장부: 사용자와 관리자가 거래 흐름을 확인할 수 있는 투명한 기록

## 기술 스택

- Next.js
- TypeScript
- SQLite
- Prisma
- Tailwind CSS

## 실행 방법

```bash
npm install
npm run db:migrate -- --name init
npm run db:seed
npm run dev
```

브라우저에서 `http://127.0.0.1:3000`으로 접속합니다.

## 보안 구현

- 비밀번호는 `bcryptjs`로 해시 저장
- 세션 토큰은 HttpOnly 쿠키와 서버 DB 해시로 관리
- 관리자 페이지는 관리자 권한만 접근
- 차단 사용자는 로그인, 상품 등록, 문의 제한
- 차단 사용자는 송금 제한
- 송금은 서버 트랜잭션으로 잔액 차감과 입금을 함께 처리
- 서버 액션에서 입력 길이, 가격 범위, URL 형식, enum 값을 검증
- Prisma 쿼리로 SQL Injection 위험 완화
